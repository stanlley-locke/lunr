# Lunr Mobile App

The Flutter frontend for Lunr, designed with a premium UI and real-time chat capabilities.

## Quick Start
1.  **Install Flutter**
    Ensure you have Flutter 3.x installed.
2.  **Get Dependencies**
    ```bash
    flutter pub get
    ```
3.  **Run App**
    ```bash
    flutter run
    ```
