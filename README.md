# Lunr Project

Lunr is a modern, real-time chat application connecting a Flutter mobile frontend with a robust Django backend.

## Project Structure
- **[lunr/](lunr/)**: Flutter Mobile Application.
- **[backend/](backend/)**: Django REST API & WebSockets.

## Features
- Real-time Messaging (Socket.IO).
- Group Chats & Management.
- Media Sharing (Images/Files).
- Contact Management.
- Chat Archiving & History.
- User Presence (Online/Typing).

## Getting Started
Please refer to the specific READMEs for each component:
- **[Backend Setup](backend/README.md)**
- **[Mobile App Setup](lunr/README.md)**

## System Requirements
- Python 3.10+
- Flutter 3.x
- PostgreSQL / SQLite
- Redis (optional)
